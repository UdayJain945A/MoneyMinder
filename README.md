## 📖 Overview

MoneyMinder is a modern, intuitive, and feature-rich Android application designed to help you take control of your personal finances. Built with Jetpack Compose and following clean architecture principles, it offers a seamless experience for tracking transactions, managing budgets, and reaching your savings goals.

---

## ✨ Features

* 📊 **Comprehensive Dashboard**
  Get a bird's-eye view of your balance, monthly income, and expenses at a glance.

* 🎙️ **Voice Commands**
  Add transactions hands-free using natural language processing
  *(e.g., "Spent 50 dollars on dinner")*

* 🎯 **Savings Goals**
  Set monthly savings targets and track your progress with real-time indicators.

* 📅 **Budget Management**
  Define monthly budget limits and receive warnings as you approach your threshold.

* 📉 **Visual Analytics**
  Understand your spending patterns with interactive weekly charts.

* 🔥 **No-Spend Streaks**
  Gamify your savings by tracking consecutive days without unnecessary spending.

* 🌗 **Dark Mode Support**
  Beautiful UI that adapts to system theme.

* 🔒 **Local-First Privacy**
  Your data stays on your device using Room database.

## 🛠️ Tech Stack

* **UI**: Jetpack Compose
* **Architecture**: Clean Architecture + MVVM
* **Dependency Injection**: Hilt
* **Database**: Room
* **Async**: Kotlin Coroutines & Flow
* **Navigation**: Navigation Compose
* **Charts**: Vico
* **Build System**: Gradle Version Catalog (TOML)

## 🚀 Getting Started

### 📋 Prerequisites

* Android Studio Ladybug or newer
* JDK 17+
* Android device/emulator (API 26+)

---

### ⚙️ Installation

```bash
git clone https://github.com/your-username/MoneyMinder.git
```

1. Open the project in Android Studio
2. Wait for Gradle sync
3. Run the app 🚀

---

## 🏗️ Project Architecture

This project follows **Clean Architecture with MVVM**, ensuring:

* Separation of concerns
* Scalable code structure
* Easy testing and maintenance

### 📂 Layers

* **Presentation Layer** → UI + ViewModel
* **Domain Layer** → Business logic
* **Data Layer** → Room DB + Repository
