# MoneyMinder 💰

MoneyMinder is a modern, intuitive, and feature-rich Android application designed to help you take control of your personal finances. Built with Jetpack Compose and following clean architecture principles, it offers a seamless experience for tracking transactions, managing budgets, and reaching your savings goals.

## ✨ Features

- **📊 Comprehensive Dashboard**: Get a bird's-eye view of your balance, monthly income, and expenses at a glance.
- **🎙️ Voice Commands**: Add transactions hands-free using natural language processing (e.g., "Spent 50 dollars on dinner").
- **🎯 Savings Goals**: Set monthly savings targets and track your progress with real-time visual indicators.
- **📅 Budget Management**: Define monthly budget limits and receive warnings as you approach your threshold.
- **📉 Visual Analytics**: Understand your spending patterns with interactive weekly spending charts.
- **🔥 No-Spend Streaks**: Gamify your savings by tracking consecutive days without non-essential spending.
- **🌗 Dark Mode Support**: Beautifully crafted UI that respects your system's theme settings.
- **🔒 Local-First**: Your financial data stays on your device using Room database for maximum privacy.

## 🛠️ Tech Stack

- **UI**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Architecture**: Clean Architecture with MVVM
- **Dependency Injection**: [Hilt](https://dagger.dev/hilt/)
- **Database**: [Room](https://developer.android.com/training/data-storage/room)
- **Asynchronous Programming**: [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html) & [Flow](https://kotlinlang.org/docs/flow.html)
- **Navigation**: [Jetpack Navigation Compose](https://developer.android.com/jetpack/compose/navigation)
- **Charts**: [Vico](https://github.com/patrykandpatrick/vico)
- **Dependency Management**: Gradle Version Catalog (TOML)

## 🚀 Getting Started

### Prerequisites

- Android Studio Ladybug or newer
- JDK 17+
- Android Device/Emulator with API level 26 (Oreo) or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/MoneyMinder.git
   ```
2. Open the project in Android Studio.
3. Wait for Gradle sync to complete.
4. Run the app on your device or emulator.
